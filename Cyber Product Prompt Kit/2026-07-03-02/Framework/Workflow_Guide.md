# Workflow Guide

## Create a New Product Document

1. Start a new ChatGPT session.
2. Upload `Framework/Master_Prompt.md`.
3. Upload `Framework/Product_Documentation_Standard.md`.
4. Ask: `Create a complete product knowledge document for <Product Name>. Use current research. Generate Markdown and PDF.`
5. Review the output.
6. Save the Markdown, PDF and source register under `Products/<Vendor>/<Product>/`.
7. Commit the changes to GitHub.

## Example

Product: AWS GuardDuty

Save outputs under:

`Products/AWS/GuardDuty/`
